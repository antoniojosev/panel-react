# VR Academy

Proyecto React + Vite con estructura modular y buenas prácticas.

## Scripts

- npm run dev
- npm run build
- npm run preview
- npm run lint

## Estructura
- src/components: UI atómico
- src/features: funcionalidades por dominio
- src/layouts: layouts de páginas
- src/pages: vistas
- src/routes: enrutamiento
- src/store: estado global
- src/services: API / infra
- src/utils: helpers
- src/types: tipos
